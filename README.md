# dev-java-cli-maven-hbase-client

## Description
A POC for connecting to a hadoop cluster using hbase.

## Tech stack
- java
- maven
  - hbase
  - zookeeper
  - hadoop

## Docker stack
- oddpoet/hbase-cdh5
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
